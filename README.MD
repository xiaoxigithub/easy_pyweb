## 试一试
```python
from easy_pyweb.wb_helpers import WebHelper

def open_page():
    wb = WebHelper(cookie_path='cookies.txt')
    page = wb.new_page()
    page.goto('https://tool.lu/timestamp/')
    print(page.title())
    assert page.title() == '时间戳(Unix timestamp)转换工具 - 在线工具'
```
